# Asteroids

This project is a modern take on the classic Asteroids arcade game, developed using TypeScript and React. Players navigate a spaceship, dodging and shooting asteroids while progressing through levels.

## Features

- Classic Asteroids gameplay with a modern twist.
- Progress through levels, each with increasing difficulty.
- Health system with heart icons representing the player's lives.
- Responsive canvas rendering for a smooth gaming experience.

## Getting Started

To get a local copy up and running, follow these simple steps.

### Prerequisites

- npm
  ```sh
  npm install npm@latest -g
  ```

### Installation

1. Clone the repo

   ```sh
   git clone https://github.com/andrew87e/octo-asteroids.git
   ```

2. Install NPM packages

   ```sh
   npm install
   ```

3. Start the development server
   ```sh
   npm start
   ```

Navigate through the game using keyboard controls. Avoid asteroids and shoot them down to progress through levels.
